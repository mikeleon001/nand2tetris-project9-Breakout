# Nand2Tetris Project 9 - Breakout Game

A Breakout-style game written in the Jack programming language.

Built as part of the Nand2Tetris course (The Elements of Computing Systems).

## How to Play
- **Left arrow** - move paddle left
- **Right arrow** - move paddle right
- Clear all bricks to win
- You have 3 lives

## Files
- `Main.jack` - Entry point
- `BreakoutGame.jack` - Main game loop, collision detection, score and lives
- `Paddle.jack` - Paddle drawing and movement
- `Ball.jack` - Ball drawing, movement and bouncing

## Running the Game
1. Compile the .jack files using the Nand2Tetris Jack Compiler
2. Run the output in the Nand2Tetris VM Emulator
